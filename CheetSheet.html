Cheat Sheet

World Wide Web

  • HTML (HyperText Markup Language): Language used to create web pages.
  • URL (Uniform Resource Locator): Unique address for web resources.
  • HTTP (Hypertext Transfer Protocol): Protocol for data transmission over the web.

URL (Uniform Resource Locator)

The URL is an address used to access resources on the internet.

Syntax:

scheme://host:port/path?query#fragment
  • Scheme: Protocol (e.g., http, https)
  • Host: Domain name (e.g., www.example.com)
  • Port: Optional (e.g., :80)
  • Path: Location of the resource (e.g., /path/to/resource)
  • Query: Optional parameters (starts with ?)
  • Fragment: Optional section reference (starts with #)

Web Servers

A web server delivers web content to clients (browsers) over the internet.

Function:

  • Stores, processes, and delivers web pages to clients upon request.
  • Handles HTTP/HTTPS requests and responses.
  • Executes server-side scripts and delivers dynamic content.
